# sercha tui

Launch the interactive Terminal UI for a visual search experience.

## Usage

```bash
sercha tui
```

## Description

The Terminal UI (TUI) provides a visual interface for:
- Searching documents with real-time results
- Browsing and previewing document content
- Managing sources and settings

The TUI uses the same core services as the CLI, so all configuration applies to both.

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `/` | Focus search input |
| `Enter` | Execute search |
| `↑/↓` | Navigate results |
| `Enter` | View selected document |
| `o` | Open document in external app |
| `q` | Quit |
| `?` | Show help |

## Features

- **Real-time search** - Results update as you type
- **Document preview** - View content without leaving the app
- **Source browser** - Manage sources and documents
- **Settings panel** - Configure search modes and providers
